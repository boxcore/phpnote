###类型运算符
类型运算符instanceof可以用来判断指定对象是否来自指定的类:

```php
<?php
class A{}
$a = new A();
var_dump($a instanceof A);
?>
```


###执行运算符
执行运算符 `命令` 内的字符串会被当做DOS命令或shell命令来执行,例如:

```php
$a = `ls`;
echo $a;
```
